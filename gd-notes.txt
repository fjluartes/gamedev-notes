Godot
    - GDScript or C#
    
Node
- building blocks for creating games in Godot.
e.g. Display an image, play animation, 3D model, etc.
- Contains collection of properties to customize behavior
- Nodes are organized into a tree structure

Scene
- Group of nodes collected into a tree
- Used to create and organize various game objects in a project
    
Signals
- Functions that connect different game objects
